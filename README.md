# Memo

一个基于 .NET 8 和 Avalonia UI 的 Windows 桌面备忘录应用。采用自定义无边框圆角窗口设计，支持系统托盘、全局快捷键、窗口置顶、长按拖拽重排，以及把单条备忘录拖出成独立弹窗。备忘录与设置保存在本机 JSON 文件中。

## 功能特性

### 备忘录管理

- 新增、编辑、删除与重排备忘录
- 输入框支持多行：Enter 保存，Shift + Enter 换行
- 双击卡片进入编辑态，保存后自动移到列表顶部
- 输入内容与已有备忘录相同时，会提升到首位而非重复新增
- 编辑中的卡片会高亮显示
- 首行为空时忽略提交
- 空列表显示引导卡片，并展示备忘录计数徽章

### 拖拽与弹窗

- 长按卡片（约半秒）后拖拽重排；拖拽期间集合保持不变
- 拖拽时显示浮空卡片、占位框，相邻卡片自动让位动画
- 靠近列表边缘时自动滚动
- 把卡片拖出主窗口后，会在释放位置弹出独立的备忘录弹窗
- 弹窗支持双击编辑（Esc 提交保存）、单独置顶、关闭过渡动画
- 弹窗时间戳可在「相对时间」和「完整时间」之间点击切换
- 开启「重复便签」前，拖出已有备忘录会移动其弹窗位置而非新建

### 窗口与标题栏

- 自定义无边框圆角窗口，支持拖动移动、最小化、关闭和置顶
- 置顶按钮带有旋转图标过渡动画
- 主窗口与弹窗的显示/隐藏带透明度 + 缩放过渡动画
- 右上角设置、置顶、最小化、关闭按钮

### 系统托盘

- 右键托盘图标打开自定义托盘菜单（显示在任务栏工作区角落）
- 左键双击托盘图标恢复/显示主窗口
- 托盘菜单提供：打开窗口、新建备忘录、切换置顶状态、退出应用
- 主窗口最小化 / 关闭均可隐藏到托盘，不占用任务栏

### 全局快捷键（均可配置）

- 切换置顶（作用目标：主窗口或最近激活的备忘录弹窗）
- 最小化到托盘
- 显示主窗口
- 快速添加：把剪贴板内容快速新增为备忘录（可单独启用/禁用，自动去重）

### 设置

- 自定义关闭按钮行为：最小化到托盘 / 直接退出（首次关闭时弹窗询问）
- 配置四项全局快捷键：点击捕获按键，按 Esc 留空，带系统保留键校验
- 启用/禁用「快速添加剪贴板内容」
- 启用/禁用「重复便签」
- 一键重置为默认设置（带二次确认）

### 数据存储

- 备忘录与设置分别保存为 `%AppData%/Memo/memos.json` 与 `settings.json`
- JSON 采用 camelCase 与缩进输出，异步保存并使用信号量保证并发安全
- 加载失败静默回退为默认值；删除文件即可在下次启动时重置

## 技术栈

- .NET 8（目标框架 `net8.0-windows`）
- Avalonia UI 11.1（默认启用编译绑定）
- C#

## 运行环境

需要 Windows 与 .NET 8 SDK。

```bash
dotnet --version
```

## 默认快捷键

| 功能 | 默认快捷键 |
| --- | --- |
| 切换置顶 | Ctrl + Alt + T |
| 最小化到托盘 | Ctrl + Alt + M |
| 显示主窗口 | Ctrl + Alt + N |
| 快速添加（剪贴板） | Ctrl + Alt + C |

快捷键可在设置窗口中修改，按 Esc 可清空某个快捷键。

## 数据存储

应用会在用户 AppData 目录下创建 `Memo` 文件夹：

```text
%AppData%/Memo/
```

主要文件：

- `memos.json`：备忘录列表
- `settings.json`：关闭行为、快捷键、快速添加、重复便签等设置

删除这些文件后，应用会在下次启动时使用空数据和默认设置重新创建。

## 使用说明

1. 在顶部输入框输入内容。
2. 按 Enter 保存为一条备忘录。
3. 按 Shift + Enter 在输入框中换行。
4. 双击已有备忘录可以编辑。
5. 点击卡片右上角删除按钮删除（下方卡片会上滑填充动画）。
6. 长按卡片并拖动可以调整顺序；拖出主窗口则弹成独立弹窗。
7. 点击标题栏图钉按钮可以切换窗口置顶。
8. 右键托盘图标打开托盘菜单，左键双击托盘图标显示主窗口。

## 开发说明

- 备忘录集合由 `MainViewModel` 管理，持久化逻辑在 `JsonMemoStorage` 中。
- 应用设置由 `JsonSettingsStorage` 保存，启动后异步加载并应用到主窗口和全局快捷键服务。
- 长按拖拽排序封装在 `DragReorderManager`：拖拽期间不修改集合，仅在释放时调用 `MainViewModel.MoveItem`。
- 弹窗由 `MemoPopoutWindow` 实现，`App` 维护弹窗列表并决定置顶快捷键的作用目标。
- 窗口过渡动画由 `WindowTransitionController` 统一提供，主窗口有额外一份以配合托盘显隐。
- 由于使用了 Windows 托盘和全局快捷键能力，当前项目按 Windows 桌面应用配置。若要支持其他平台，需要替换托盘、快捷键和目标框架相关实现。
